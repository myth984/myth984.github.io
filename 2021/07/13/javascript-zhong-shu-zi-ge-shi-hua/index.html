JavaScript中数字格式化
2021-07-13

需求

需要在前端页面中将某些数字

  • 用千分位形式展示
  • 保留n位小数,不足补零

解决方案

发现 numeraljs 这个包整好可以解决我的问题

于是npm install numeral

import numeral from "numeral" 
// 千分位标识 '1,000'
numeral(1000).format('0,0');
// 千分位标识并保留四位小数 '10,000.0000'
numeral(10000).format('0,0.0000');
